Drop to Paradise

Drop to Paradise é um website com sistema de gerenciamento de produtos e vendas desenvolvido com foco em e-commerce. O projeto tem como objetivo integrar a gestão de produtos em uma loja virtual, facilitando o controle de estoque, o cadastro de produtos e o gerenciamento de vendas.

Funcionalidades

- Cadastro de produtos, clientes, funcionários, marcas e categorias 
- Atualização, Relatórios, exclusão e gerenciamento de estoque
- Controle de preços, custos e descontos
- Sistema de promoção de produtos
- Organização de categorias e marcas
- Registro e gerenciamento de vendas no sistema desktop

Tecnologias Utilizadas

- C# (Windows Forms)
- PHP 
- XAMPP – ambiente de desenvolvimento para execução do servidor local (Apache e MySQL)
- HTML, CSS e JavaScript 
- BR Modelo
- Adobe Photoshop 

Objetivo do Projeto

O projeto foi desenvolvido como parte dos estudos em desenvolvimento de software, com o objetivo de aplicar conhecimentos de programação, banco de dados e desenvolvimento de sistemas para a conclusão do curso.
